## 2. 1 트랜스포머 아키텍처란

![[Pasted image 20260617095317.png|348]]

트랜스포머 아키텍처: 인코더 + 디코더
- 현재 트랜스포머는 자연어 처리, 컴퓨터 비전, 추천 시스템 등 모든 AI 분야에서 핵심 아키텍처로 사용되고 있다.

트랜스포머는 RNN의 문제(그래디언트 소실/증폭, 입력이 길어지면 성능 저하, 학습 속도 느림)를 해결하기 위해 입력을 하나씩 순차적으로 처리하는 방식을 버리고 셀프 어텐션이라는 개념을 도입했다.
- 셀프 어텐션은 입력된 문장 내의 각 단어가 서로 어떤 관련이 있는지 계산해서 각 단어의 representation(표현)을 조정하는 역할을 한다.

## 2.2 텍스트를 임베딩으로 변환하기

![[Pasted image 20260617095546.png|224]]

- 토큰화: 텍스트를 적절한 단위로 잘라 숫자형 아이디를 부여하는 것
- 토큰 임베딩: 토큰을 토큰 임베딩 층에 통과시켜 여러 숫자의 집합인 토큰 임베팅으로 변환
- 위치 인코딩 층: 토큰 임베딩에 토큰의 위치 정보를 담고 있는 위치 임베딩으로 추가해 최종 임베딩을 만든다.

사전: 토큰이 어떤 숫자 아이디로 연결됐는 지 기록하는 것
- 사전에 없는 단어 (OOV - Out Of Vocabulary) 문제..

서브워드 토큰화 방식: 데이터에 등장하는 빈도에 따라 토큰화 단위를 결정
- 요즘은 서브워드 토큰화 방식을 많이 사용한다.
![[Pasted image 20260617095848.png|348]]


위치 인코딩: 트랜스포머가 모든 입력 토큰을 동등하게 처리하기 때문에 입력으로 위치 정보를 함께 더해야 한다.

## 2.3 어텐션 이해하기

어텐션은 사람이 단어 사이의 관계를 고민하는 과정을 딥러닝 모델이 수행할 수 있도록 모방한 연산
- 사람은 단어를 그 단어만으로 해석하지 않고 주변의 단어를 통해 맥락을 추가한 새로운 단어로 이해.

![[Pasted image 20260617164157.png|385]]

- 쿼리: 검색창에서 검색할 때, 우리가 입력하는 검색어
- 키: 쿼리와 관련이 있는지 계산하기 위해 문서가 가진 특징
- 값: 우리가 검색하면서 원하는 것

- 단어를 모두 동등하게 반영하는 평균 방법으로는 관련이 깊은 단어를 더 많이 반영하는다는 목표는 달성하지 못한다.
- 트랜스포머에서는 W_q, W_k 가중치를 통해 토큰과 토큰 사이의 관계를 계산하는 능력을 학습시켰다.

![[Pasted image 20260621132147.png|506]]

멀티 헤드 어텐션 -> 고정된 토큰 임베딩을 head만큼 자르고 w_q, w_k, w_v로 통과시켜 여러 관점으로 바라보는 임베딩을 얻는다!

어텐션은 각 단어 토큰의 임베딩을 “그 단어 하나만의 고정된 의미”로 두지 않고, 문장 안의 다른 단어들과 얼마나 관련 있는지를 계산해서 **문맥이 반영된 새로운 임베딩**으로 바꾸는 연산! (by codex)

## 2.4 정규화와 피드 포워드 층

딥 러닝 모델에 데이터를 입력할 때, 입력 데이터의 분포가 서로 다르면 모델의 학습이 잘 되지 않기 때문에 데이터를 정규화하여 입력하는 것이 중요하다.

정규화: norm_x = (x - 평균) / 표준편차

딥러닝에서는 평균과 표준편차를 구할 데이터를 어떻게 묶는 지에 따라 크게 배치 정규화와 층 정규화로 구분한다.
- 일반적으로 이미지는 배치 정규화를, 자연어 처리에서는 층 정규화를 사용한다.

층 정규화: 각 토큰의 임베딩 벡터 안에서 값들의 평균과 분산을 정리해서, 학습이 안정적으로 되게 만드는 연산(by codex)

피드 포워드 층: 데이터의 특징을 학습하는 완전 연결 층(Fully Connected Layer)
- 멀티 헤드 어텐션이 단어 사이의 관계를 파악하는 일이라면 피드 포워드 층은 입력 텍스트 전체를 이해하는 역할을 담당한다.

![[Pasted image 20260630150110.png|247]]

인코더와 디코더는 크게 두 가지의 차이점이 있다
- 마스크 어텐션 
- 크로스 어텐션

트랜스포머 모델은 앞에서 생성한 토큰을 기반으로 다음 토큰을 생성한다. 이렇게 순차적으로 생성해야하는 특징을 인과적(causal) 또는 자기-회귀적(auto-regressive)이라고 말한다.
![[Pasted image 20260630155706.png]]
-inf에 softmax를 취하면 0이 나온다! -> 대각선 위는 모두 0

크로스 어텐션 -> 인코더 결과를 디코더에서 활용
- 영어에서 한국어로 번역한다고 했을 때 인코더가 영어 문장을 받아 처리한 결과를 한국어를 생성하는 디코더가 받아 활용
